*Use these when you need the model to understand a specific type style or treatment.*

### condensed-slab-inline-extrude.jpg
**What it shows:** Unispare condensed slab serif — full character set with three
variants: Regular, Inline (hairline channel cut through main stems), and Extrude
(block shadow offset). Ligature system shown at bottom. Navy + gold + cream palette.

**Primary use:** Headline type reference for any heritage workwear, athletic, or
western badge. The inline variant is exactly right for Western Welder.
**Style profiles:** Heritage Americana, Vintage Athletic, Western/Ranch
**Reference strength:** STRONG — send to model when you want condensed slab with
inline detail

### western-slab-serif-mixed-case.jpg
**What it shows:** Elmville slab serif — large mixed case display with bold flat-foot

serifs at this weight. Star ornament as punctuation device between words.
High contrast red field with white type — clean, bold, legible.
**Primary use:** When a job calls for bold slab at display scale without all-caps.

**Reference strength:** STRONG — good model input for slab serif scale and weight
